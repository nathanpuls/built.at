A simple, verified link from your website

You already own your site — this just extends it. Built.at lets you publish a verified link straight from your domain name using nothing but a DNS TXT record. No hosting. No database. If you own the domain, you own the link.

How It Works

Add one line to your DNS, and you’ve got a trusted, branded link that points anywhere (like your social media, link page, documentation, or any other related resource).

1. Log in to your domain provider

Where your website is hosted

Cloudflare SquareSpace GoDaddy NameCheap BlueHost

2. Open your DNS records

3. Add a TXT record

Use @ as the host name

4. Set the value

"built.at=anywhere LINK"
Wherever you want your built.at link to point to. Use the full URL, include http(s), up to 255 characters. (Cloudflare adds quotes automatically)

Link Formatting

For any link built.at/[domain] if it’s a .com, you can skip the extension. Other domains must include it.

Dot Com Example

Extension not required, both formats work

built.at/linksaw

built.at/linksaw.com

Non Dot Com Example

Extension required

built.at/built.at
